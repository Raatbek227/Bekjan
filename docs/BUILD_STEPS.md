# Build Steps

1. Scaffold project architecture.
2. Connect PostgreSQL and run Prisma migrations.
3. Seed roles, service categories, services, product categories, and demo products.
4. Complete authentication flows: JWT, refresh tokens, email verification, reset password, Google OAuth.
5. Build store: product listing, product details, cart, wishlist, checkout, payments.
6. Build booking: availability, staff assignments, working hours, admin calendar.
7. Build dashboards: customer dashboard and admin CRM.
8. Add CMS: blog, gallery, reviews, contacts, SEO settings.
9. Add notifications, loyalty tiers, global search, and multilingual content.
10. Harden production deployment: logging, monitoring, caching, rate limits, backups.
